# DEAL Radar

eBay Deal Finder for Flipping Authenticated Collectibles

Find underpriced signed memorabilia, vintage vinyl, space collectibles, and art prints on eBay.

## Features

- Real-time eBay Browse API searches
- Configurable deal targets (KAWS, Shepard Fairey, Mr. Brainwash, etc.)
- Watchlist to track potential purchases
- Price filtering to find deals below market value

## Deal Categories

- **Art**: KAWS, Shepard Fairey, Mr. Brainwash, Death NYC, Banksy
- **Space**: Apollo 11, Buzz Aldrin, Neil Armstrong signed photos
- **Music**: Signed vinyl, pickguards, memorabilia with COA
- **Bearbrick**: 1000% collectible figures

## Setup

1. Clone the repo
2. Copy `.env.example` to `.env` and add your eBay API credentials
3. Run `pip install -r requirements.txt`
4. Run `python app.py`
5. Open http://localhost:5051
