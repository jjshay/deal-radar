# DEAL Radar - Flipping Strategy Guide

## Overview

DEAL Radar finds underpriced authenticated collectibles on eBay that can be purchased and resold for profit. The system targets items where sellers don't know the true market value.

## Authentication Requirements

### Must Have COA From:
- **JSA** (James Spence Authentication)
- **PSA/DNA** (Professional Sports Authenticator)
- **BAS** (Beckett Authentication Services)

### Red Flags - Avoid:
- No COA or "COA included" without specifying company
- Price too good to be true (<50% market value)
- Stock photos instead of actual item
- New seller with no feedback
- Ships from China/overseas (likely fake)

## Daily Workflow

### Morning Scan (8 AM)
1. Run DEAL Radar automated scan
2. Review A+ and A deals first
3. Check seller feedback and item photos
4. Verify COA authenticity (look up cert number if possible)
5. Purchase best deals before competition

### Listing (Evening)
1. Photograph purchased items
2. Research recent sold comps
3. Write detailed descriptions mentioning:
   - Authentication company + cert number
   - Artist/subject
   - Condition details
   - Provenance if known
4. List at 10-15% below market
